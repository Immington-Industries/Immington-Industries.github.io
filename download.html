Way Cooler comes in two primary flavors: i3 and awesome.

The i3 flavor is set up like a default i3 install, with the same keybindings and tiling algorithm. The only difference between this flavor and actual i3 is that the primary configuration format is still Lua.

The awesome flavor disables the i3 controls and gives Lua full control of the tiling. Just like awesome, there are customizable modes to switch between that control how windows are laid out on the screen.

We also ship with various default themes. Feel free to submit your own and we might add them to the list!

Flavor: Theme:
Background client (way-cooler-bg)
Screenshot client (wc-grab)
Lockscreen client (wc-lock)

curl https://way-cooler.github.io/way-cooler-release-i3-default.sh -sSLf | sh -s
